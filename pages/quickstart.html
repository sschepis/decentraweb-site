Quick Start Guide

Get DecentraClick running locally in minutes.

Prerequisites

  • Node.js 18+ — Required for all servers and build tools
  • npm or yarn — Package manager
  • Git — Version control
  • A Web3 wallet — MetaMask or any EIP-1193 compatible wallet

Installation

  1. Clone the repository 
    git clone https://github.com/decentraclick/decentraclick.git
cd decentraclick
  2. Install dependencies 
    npm install
  3. Configure environment 
    cp .env.example .env
# Edit .env with your RPC URLs, private keys, and API keys
  4. Start all services 
    npm run dev

    This starts the Vite frontend, Gun relay, compilation server, and assistant server concurrently.

  5. Open the app

    Navigate to http://localhost:5173 in your browser and connect your wallet.

Individual Services

You can also start services individually:

CommandServiceDescription
npm run frontendFrontendVite dev server on port 5173
npm run compile-serverCompilerSolidity compiler WebSocket server
npm run gun-relayGun RelayGun.js P2P relay node
npm run assistantAI AssistantAI assistant API gateway
npm run sumsub-serverKYC ServerSumSub KYC integration

Deploy Contracts

To deploy the platform's smart contracts to a testnet:

# Deploy to Optimism Sepolia testnet
npm run deploy:optimism-sepolia

# Deploy to Ethereum Sepolia testnet
npm run deploy:sepolia

# Deploy to Optimism mainnet (production)
npm run deploy:optimism
⚠
Mainnet deployment requires real ETH/OP for gas. Always test on Sepolia first.

Build for Production

# Build Solidity contracts
npm run forge:build

# Build frontend
npm run build

# Run tests
npm run test:run

# Lint code
npm run lint
