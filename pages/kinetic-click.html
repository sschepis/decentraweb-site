KineticClick — B2B Outbound Lead Engine

AI-driven outbound lead generation with kinetic energy scoring, signal-based prioritization, AegisPipe data pipelines, and automated outreach campaigns.

💡
KineticClick is a standalone B2B application within the platform. All leads, signals, campaigns, and pipeline data are stored in Gun.js P2P storage with on-chain anchoring.

Core Concepts

  • Kinetic Energy Score — A 0-100 score computed per lead from buying signals. Signals compound: 9 signal types (job changes, funding rounds, tech stack changes, hiring, product launches, partnerships, expansions, regulatory events, intent data) each contribute weighted points. Recency, source authority, and C-level executive signals boost the score
  • AegisPipe Data Pipeline — A Zod-validated, AST-based data transformation pipeline (repurposed from the code transformation engine). Steps include: schema validation, deduplication, normalization, email verification, scoring, filtering, signal tagging, and waterfall retrieval. Supports dry-run and apply modes with full rollback
  • "Truth Wizard" Workflow — A 5-stage pipeline: Signal Acquisition → Truth Verification → Consensus Scoring → Approval → Execution

Pipeline Management

Leads progress through configurable pipeline stages with automation rules:

  • Pipeline Builder — Visual drag-and-drop pipeline stage editor. Define stages, transition criteria, and automation actions
  • Lead Table — Sortable, filterable lead list with kinetic energy scores, signal counts, and stage indicators
  • Batch Approval — Review and approve/reject lead batches with bulk actions (assign, export, enrich)
  • Signal Cards — Per-lead signal visualizations showing which signals are active and their recency

AegisPipe Data Pipeline

The data processing engine that powers lead enrichment and scoring:

// Zod-validated pipeline configuration
const pipeline = {
  mode: 'apply', // or 'dry_run'
  steps: [
    { type: 'validate_schema', schema: LeadSchema },
    { type: 'deduplicate', keys: ['email', 'company'] },
    { type: 'normalize', mappings: { company: 'titleCase' } },
    { type: 'verify_email', service: 'email-verification' },
    { type: 'score', weights: signalWeights },
    { type: 'filter', minScore: 30 },
    { type: 'tag_signals', signalDefs: [...buyingSignals] },
    { type: 'waterfall', tiers: ['hot', 'warm', 'cold'] }
  ]
};
