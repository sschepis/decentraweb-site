App Store & Builder

30+ pre-built apps, a visual app builder, and a full app marketplace for custom applications.

Overview

DecentraClick includes an integrated App Store with 30+ pre-built applications and an App Builder for creating custom apps. Organizations can install apps from the store or build their own.

App Store

The App Store provides discovery and installation of applications. Apps are categorized by function (Finance, Governance, Social, Developer, etc.) and can be installed per-domain.

  • 30+ Pre-built Apps — Ready-to-use applications for common organizational needs
  • Per-domain Installation — Each organization installs only the apps they need
  • App Categories — Finance, Governance, Social, Developer, Commerce, Content, and more
  • Version Management — Track and update installed app versions

App Builder

The visual App Builder lets users create custom applications with:

  • Template Selection — Start from pre-built templates or from scratch
  • Visual Editor — Drag-and-drop interface for app layout
  • Code Editor — Monaco-based code editor for custom logic
  • Service Integration — Connect to any platform service (database, compute, etc.)
  • Deployment — Deploy apps to your domain or publish to the App Store
  • Security Settings — Configure permissions and access controls

App Architecture

Each app is a React component that receives platform context and can use all hooks and services. Apps run within the platform's security sandbox with configurable permissions.
