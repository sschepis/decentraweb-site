Smart Contracts

A comprehensive on-chain registry system with 36+ contracts deployed on Optimism.

Overview

DecentraClick deploys a comprehensive set of smart contracts to Optimism that serve as the on-chain backbone of the platform. These contracts manage registries, identity, tokens, governance, and more.

Registry Contracts

The platform uses a registry pattern where each domain-specific concern has its own on-chain registry contract:

Contract Deployment

Contracts are deployed using the CLI deployment script. The deployed addresses are stored in public/deployed-contracts.json for the frontend to reference.

# Deploy to testnet
npm run deploy:optimism-sepolia

# Deploy to mainnet
npm run deploy:optimism

ABI Management

Contract ABIs are stored as TypeScript files in src/contracts/abi/ with 52+ ABI definitions. The forge:artifacts script compiles Solidity and extracts ABIs automatically.
