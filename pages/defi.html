Composable DeFi

Flash loans, fee management, variable pricing, and collateral token factories for DeFi strategies.

Overview

DecentraClick provides composable DeFi primitives that organizations can use to build financial products and strategies. These are implemented as Diamond facets and coordinated through the composableDefiService.

DeFi Primitives

  • Flash Loans — Borrow and repay within a single transaction via the FlashLenderFacet
  • Fee Management — Configure fee structures with the FeeManagerFacet; distribute fees to stakeholders via FeeDistributorFacet
  • Variable Pricing — Dynamic pricing curves (bonding curves, Dutch auctions) via VariablePriceFacet
  • Collateral Tokens — Create restricted collateral tokens via CollateralTokenFactoryFacet
  • Token Launchpad — Launch and distribute new tokens with configurable sale parameters
  • Marketplace — Peer-to-peer trading marketplace with escrow integration

Strategy Builder

The Composable DeFi app allows users to visually compose DeFi strategies by chaining primitives together. Strategies can combine flash loans, swaps, and fee distributions into automated workflows.
