NFT Studio

Create and manage NFT collections with Diamond proxy, metadata generation, and minting policies.

Overview

NFT Studio provides a complete toolkit for creating, managing, and distributing NFT collections. Collections are deployed as Diamond contracts with ERC-721 and ERC-1155 facets, enabling rich on-chain functionality.

Features

  • Collection Creation — Deploy new NFT collections with customizable parameters
  • Metadata Generation — On-chain SVG templates and off-chain IPFS metadata
  • Minting Policies — Open minting, whitelist-only, or permissioned minting
  • Batch Operations — Mint, transfer, and manage multiple tokens at once
  • Marketplace Integration — List NFTs on the built-in marketplace
  • Royalties — Configure creator royalties on secondary sales
  • Hierarchical NFTs — ERC-6059 support for parent-child NFT relationships

Token Standards

ERC-721

Unique non-fungible tokens (standard NFTs)

ERC-1155

Multi-token standard (editions, fungible + non-fungible)

ERC-6059

Hierarchical/nestable NFTs

nftStudioService

NFT collection creation and management (20KB)

NftStudioRegistry

On-chain collection registry

SVGTemplatesFacet

On-chain SVG metadata generation

NftStudioApp

Pre-built NFT collection builder
