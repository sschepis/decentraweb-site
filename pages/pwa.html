PWA & Offline

Progressive Web App with service workers, push notifications, offline sync, and biometric auth.

Overview

DecentraClick is a full Progressive Web App (PWA) that works offline, supports push notifications, and can be installed on any device. The offline-first architecture ensures the app remains functional even without internet connectivity.

Features

  • Service Worker — Full lifecycle management with multiple caching strategies
  • Offline Sync Queue — Operations performed offline are queued and synced when connectivity returns
  • Push Notifications — Receive notifications for proposals, transactions, messages, and more
  • Install Prompt — Native install experience on mobile and desktop
  • Biometric Auth — Fingerprint/Face ID authentication bridge
  • Background Sync — Data synchronization continues in the background

Caching Strategies

StrategyBehaviorBest For
Cache FirstServe from cache, fallback to networkStatic assets, fonts, images
Network FirstTry network, fallback to cacheAPI data, dynamic content
Stale While RevalidateServe cache immediately, update in backgroundSemi-dynamic content

Offline Capabilities

When offline, users can:

  • Browse previously loaded pages and data
  • Create and edit content (queued for sync)
  • View cached transaction history
  • Access installed applications
