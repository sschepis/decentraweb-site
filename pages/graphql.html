GraphQL API

In-browser GraphQL server with 8 schema modules, type-safe resolvers, and real-time subscriptions.

Overview

DecentraClick runs a full GraphQL API server directly in the browser using GraphQL Yoga. This provides a unified query interface across all platform data sources without requiring a backend server.

Schema Modules

ModuleDescriptionExample Queries
CoreDomains, identities, rolesgetDomain, getIdentity, getRoles
WalletBalances, transactions, tokensgetBalance, getTransactions
DeFiPools, swaps, liquiditygetPools, getSwapHistory
AppsInstalled apps, app storegetInstalledApps, searchApps
InfraServices, compute, databasesgetServices, getComputeFunctions
SocialMessages, wiki, CRMgetMessages, getWikiPages
GovernanceDAO proposals, voting, delegationgetProposals, getVotingPower
StablecoinSDU vaults, reserve, health, pegsduOverview, sduVaults, sduReserve

Data Sources

  • Gun.js — P2P data queries for off-chain content
  • On-chain — Direct contract reads via Viem
  • Namespace — Scoped data access through the namespace service
  • Cache — In-memory caching with configurable TTL

Real-time Subscriptions

The GraphQL API supports subscriptions for real-time data updates, powered by the event bus. Subscribe to domain changes, new proposals, transaction confirmations, and more.
