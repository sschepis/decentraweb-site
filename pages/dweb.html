DWeb Publishing

Publish decentralized websites via IPFS and Gun.js with custom domain routing and SDK integration.

Overview

The Decentralized Web (DWeb) feature lets users publish and browse websites that are hosted on decentralized infrastructure. Sites are bundled and distributed via IPFS and Gun.js, making them censorship-resistant and always available.

Features

  • Site Publishing — Publish HTML/CSS/JS sites to IPFS with Gun.js metadata
  • Domain Resolution — Map .dnet domains to DWeb site addresses
  • DWeb Browser — Built-in browser for navigating DWeb sites
  • SDK Integration — Embedded sites can interact with the platform via the DWeb SDK
  • Site Bundling — Automatic bundling and optimization of site assets
  • Version History — Previous versions accessible via content hashes

Publishing Workflow

  1. Create Site — Build your site using the Site Publisher or upload files
  2. Bundle — Assets are bundled and optimized automatically
  3. Upload to IPFS — Bundle pinned to IPFS for content-addressed hosting
  4. Register — Map your domain to the IPFS hash via Gun.js
  5. Browse — Access via the DWeb Browser or direct resolution
