Smart Wallet

Contract-based wallets with guardian recovery, batch transactions, and session key support.

Overview

The Smart Wallet is DecentraClick's user-facing wallet experience built on ERC-4337 account abstraction. It provides a contract-based wallet with advanced security and UX features.

Features

  • Guardian Recovery — Designate trusted addresses as guardians who can help recover wallet access if you lose your keys
  • Batch Transactions — Execute multiple operations in a single transaction for lower gas and better UX
  • Session Keys — Grant dApps limited, time-bound permission to transact on your behalf
  • Gas Abstraction — Pay gas in ERC-20 tokens or have organizations sponsor your gas via paymasters
  • Multi-Wallet Support — Link multiple EOA wallets to your smart account
  • Transaction History — Full history of all UserOperations with status tracking

Recovery Process

  1. Set up guardians — Add 3-5 trusted addresses as recovery guardians
  2. Initiate recovery — If access is lost, guardians can initiate recovery
  3. Threshold approval — A configurable threshold of guardians must approve (e.g., 3 of 5)
  4. New owner set — Once threshold is met, a new owner key is set for the smart wallet
