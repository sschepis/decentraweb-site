DecentraClick Documentation

The complete guide to the decentralized application platform built on Optimism. Domains as organizations, composable services, and sovereign infrastructure.

Built on Optimism 💎 EIP-2535 Diamond 🔒 ERC-4337 🌐 Gun.js P2P ⚙ React + TypeScript

Platform Overview

DecentraClick is a multi-tenant, composable infrastructure platform where domains function as organizations. It provides a rich ecosystem of plugins, apps, and services — all built on decentralized technologies with blockchain settlement on Optimism.

💡
New to DecentraClick? Start with the Quick Start Guide to get up and running, then explore the Architecture Overview to understand how everything fits together.

Core Concepts

🌐

Domains as Organizations

Register domains that become fully-featured organizations with members, roles, treasury, governance, and subdomains.

Multi-tenantGovernance
👤

Decentralized Identity

ERC-1056 identity registry with verifiable credentials, attestations, KYC/AML integration, and multi-wallet linking.

ERC-1056KYC
💎

Diamond Proxy (EIP-2535)

Modular smart contract system with 54+ facets for tokens, DeFi, governance, identity, stablecoins, and more. Mix and match capabilities.

54+ FacetsUpgradeable
💳

Account Abstraction

ERC-4337 smart accounts with gasless transactions, paymasters, session keys, and guardian recovery.

ERC-4337Gasless
🏛

DAO Governance

Full governance framework with multiple voting strategies, proposal types, delegation, and treasury management.

ProposalsVoting
🔌

Plugin SDK

Extend the platform with sandboxed plugins. 11 permission categories, iframe isolation, and a full lifecycle API.

SandboxedExtensible

Infrastructure Services

⚙

Serverless Compute

Deploy WASM functions triggered by on-chain events, webhooks, or cron schedules with sandboxed execution.

🗃

Decentralized Database

P2P data storage powered by Gun.js with namespace isolation, CRDT conflict resolution, and offline support.

🔌

GraphQL API

In-browser GraphQL server with 8 schema modules, type-safe resolvers, caching, and real-time subscriptions.

🌐

DWeb Publishing

Publish decentralized websites via IPFS and Gun.js with custom domain routing and SDK integration.

📱

PWA & Offline

Progressive Web App with service workers, push notifications, offline sync, biometric auth, and install prompts.

📣

Event Bus

Unified pub/sub system across local, P2P (Gun.js), and on-chain events with 30+ typed channels.

Financial & Business

📈

Composable DeFi

Flash loans, fee management, variable pricing, and collateral token factories for building DeFi strategies.

🤝

Escrow Service

Milestone-based escrow with dispute resolution, arbitration, time-locked release, and evidence workflows.

📦

Trade Finance

Deal lifecycle management with approval queues, token issuance, and supply chain integration.

🎨

NFT Studio

Create and manage NFT collections using the Diamond proxy with metadata generation and minting policies.

💲

SDU Stablecoin (ARE)

Dollar-pegged stablecoin with overcollateralized vault minting, 4-sleeve reserve, health-driven state machine, and autonomous revenue waterfall.
